About my custom markdown flavour

If you like the format of these tutorials, you might be wondering which tool I used. What you see is tutorial-markdown. It's a tool developed by me, specifically to allow me for rapid and easy writing of tutorials. If you're interested in getting access to it, contact me here (TODO: add contact forms)

--- markdown

You need to use `--- markdown` to exit questions mode and switch back to markdown mode.
There are many different modes. Another one is buttons! Notice that buttons are not just 
normal links. *tutorial-markdown* treats them specially. It is able to build
dependency graphs between different posts and then assemble them into a
graph that looks like a skill-tree.
